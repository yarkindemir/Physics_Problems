<h2>4. Phase Difference</h2>

<p><strong>Given:</strong></p>
<p>The distance between the two points is &lambda; / 3.</p>

<p><strong>Formula:</strong></p>
<p>&Delta;&phi; = 2&pi; (&Delta;x / &lambda;)</p>

<p><strong>Substitute:</strong></p>
<p>&Delta;&phi; = 2&pi; ((&lambda; / 3) / &lambda;)</p>

<p>&Delta;&phi; = 2&pi; / 3</p>

<p><strong>Answer:</strong></p>
<ul>
  <li>Phase difference = 2&pi;/3 radians</li>
</ul>
