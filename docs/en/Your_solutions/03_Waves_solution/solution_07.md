<h2>7. Standing Wave Modes</h2>

<p><strong>Given:</strong></p>
<ul>
  <li>Length of the string: L = 80 cm = 0.80 m</li>
  <li>The standing wave has 4 antinodes</li>
</ul>

<p><strong>Important idea:</strong></p>
<p>
In a standing wave on a string fixed at both ends, each antinode corresponds to
half of a wavelength.
So if there are n antinodes, the length of the string is:
</p>

<p>L = n&lambda; / 2</p>

<p>Here, the number of antinodes is:</p>
<p>n = 4</p>

<p><strong>Step 1: Write the formula</strong></p>
<p>L = n&lambda; / 2</p>

<p><strong>Step 2: Substitute the known values</strong></p>
<p>0.80 = 4&lambda; / 2</p>

<p><strong>Step 3: Simplify</strong></p>
<p>4&lambda; / 2 = 2&lambda;</p>

<p>So:</p>
<p>0.80 = 2&lambda;</p>

<p><strong>Step 4: Solve for wavelength</strong></p>
<p>&lambda; = 0.80 / 2</p>

<p>&lambda; = 0.40 m</p>

<p><strong>Final Answer:</strong></p>
<ul>
  <li>The wavelength is 0.40 m.</li>
</ul>

<p><strong>Short explanation:</strong></p>
<p>
A standing wave with 4 antinodes means the string contains 4 half-wavelengths.
That is why the total length is equal to 2 full wavelengths, so the wavelength is 0.40 m.
</p>
