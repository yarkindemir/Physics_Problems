<h2>8. Waves</h2>

<p><strong>Goal:</strong></p>
<p>
Determine which functions can represent a traveling wave.
</p>

<p><strong>Important idea:</strong></p>
<p>
A traveling wave has the general form:
</p>

<p>y(x,t) = f(x - vt) &nbsp;&nbsp;or&nbsp;&nbsp; y(x,t) = f(x + vt)</p>

<p>
This means the wave shape moves without changing form.
</p>

<hr>

<p><strong>a) y(x,t) = A cos(kx² - &omega;t)</strong></p>

<p>
This function contains <strong>x²</strong>, not just x.
So it is NOT in the form f(x ± vt).
</p>

<p>
Because of this, the shape of the wave would change with time,
so it is not a standard traveling wave.
</p>

<p><strong>Conclusion:</strong> Not a traveling wave</p>

<hr>

<p><strong>b) y(x,t) = A(x - vt)²</strong></p>

<p>
This is exactly in the form:
</p>

<p>y = f(x - vt)</p>

<p>
Even though it is not sinusoidal, it still represents a shape
that moves without changing form.
</p>

<p><strong>Conclusion:</strong> Traveling wave ✔️</p>

<hr>

<p><strong>c) y(x,t) = A log(x + vt)</strong></p>

<p>
This is in the form:
</p>

<p>y = f(x + vt)</p>

<p>
So it also represents a wave moving in the opposite direction.
</p>

<p>
Note: The logarithm requires (x + vt) &gt; 0 to be defined.
</p>

<p><strong>Conclusion:</strong> Traveling wave ✔️</p>

<hr>

<p><strong>Final Answer:</strong></p>
<ul>
  <li>(a) ❌ Not a traveling wave</li>
  <li>(b) ✔️ Traveling wave</li>
  <li>(c) ✔️ Traveling wave</li>
</ul>
