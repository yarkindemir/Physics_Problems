<h2>2. String Harmonics</h2>

<p><strong>Given:</strong></p>
<ul>
  <li>String length: L = 64 cm = 0.64 m</li>
  <li>Fundamental frequency: f = 330 Hz</li>
</ul>

<p><strong>For the fundamental mode:</strong></p>
<p>L = &lambda; / 2</p>

<p>&lambda; = 2L = 2 × 0.64 = 1.28 m</p>

<p><strong>Wave speed formula:</strong></p>
<p>v = f &times; &lambda;</p>

<p>v = 330 × 1.28 = 422.4 m/s</p>

<p><strong>Answer:</strong></p>
<ul>
  <li>Wave speed = 422.4 m/s</li>
</ul>
