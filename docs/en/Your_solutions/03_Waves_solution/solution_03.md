<h2>3. Superposition Principle</h2>

<p><strong>Given:</strong></p>
<p>y₁(x,t) = A sin(kx − ωt)</p>
<p>y₂(x,t) = A sin(kx + ωt)</p>

<p><strong>Superposition:</strong></p>
<p>y = y₁ + y₂</p>

<p><strong>Use identity:</strong></p>
<p>sin(a) + sin(b) = 2 sin((a + b)/2) cos((a − b)/2)</p>

<p><strong>Apply:</strong></p>
<p>y = 2A sin(kx) cos(ωt)</p>

<p><strong>Standing wave equation:</strong></p>
<p>y(x,t) = 2A sin(kx) cos(ωt)</p>

<p><strong>Nodes (positions where y = 0):</strong></p>
<p>sin(kx) = 0</p>
<p>kx = nπ</p>
<p>x = nπ / k</p>

<p>Since k = 2π / λ:</p>
<p>x = nλ / 2</p>

<p><strong>Answer:</strong></p>
<ul>
  <li>Standing wave: y(x,t) = 2A sin(kx) cos(ωt)</li>
  <li>Nodes: x = nλ/2</li>
</ul>
